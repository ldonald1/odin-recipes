This is a recipe website that contains recipes for a variety of different dishes. The site is made with HTML and currently consists of four different pages: 

- index.html which acts as the home page and provides links to each recipe 
- chicken-katsu.html which contains the recipe for chicken katsu 
- banana-pancakes.html which contains the recipe for banana pancakes 
- berry-smoothie.html which contains the recipe for a berry smoothie.

The skills demonstrated in this project include:

- unordered and ordered lists
- images
- links
- elements and tags
- attributes
- HTML boilerplate
- git add, commit, push
